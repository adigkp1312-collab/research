# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for the Adiyogi LangChain project.

## What is an ADR?

An ADR is a document that captures an important architectural decision made along with its context and consequences.

## Creating a New ADR

1. Copy `000-template.md`
2. Rename to `NNN-title.md` (next number in sequence)
3. Fill in all sections
4. Update this README index
5. Get approval before implementing

## Status Definitions

- **Proposed**: Under discussion, not yet approved
- **Accepted**: Approved and should be followed
- **Deprecated**: No longer relevant, kept for history
- **Superseded**: Replaced by another ADR (link to new one)
