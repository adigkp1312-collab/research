# Architecture Decision Records (ADR)

This directory contains Architecture Decision Records for the Adiyogi LangChain project.

## What is an ADR?

An ADR documents a significant architectural decision along with its context and consequences.

## Creating a New ADR

1. Copy `000-template.md`
2. Name it `NNN-title.md` (next number in sequence)
3. Fill in all sections
4. Update this README index
5. Get approval before implementing
